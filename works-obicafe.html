Obi Cafe(架空)

URL

obicafe_webdesign/index.html

概要

自身の課題として制作した架空カフェサイトになります。
当サイトは、自家焙煎のこだわり珈琲、隠れ家カフェをテーマにし、作成しました。

サイト制作の目的

サイトの構成やデザイン / コーディング、レスポンシブデザインについで学習することを目的として制作しました。

ターゲット

自家焙煎したこだわり珈琲に興味を持っている人々。
一風変わった雰囲気を求め、普段とは違う体験をしたいと思っている人々。

デザイン

私の地元、熊本にある帯山から名前を取っており、路地裏にある隠れ家カフェというテーマから、
茶色などの落ち着いた色調で制作しました。

制作範囲

デザイン / コーディング / バナー作成

制作期間

１ヶ月
2021年9月制作

使用ツール

XD / Visual Studio Code

使用言語

HTML / CSS
